# vlogvibemain

VlogVibe - Premium Content Platform

## Features

- Premium vlogging platform with high-quality content
- User authentication and profiles
- Direct messaging with creators
- Smooth animations with GSAP

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the app:
   ```bash
   npm run dev
   ```
